# SFX (правильно / неверно)

Чтобы включить звуковые эффекты ответа, положи файлы:

- `public/sfx/correct.mp3`
- `public/sfx/wrong.mp3`

Формат `mp3` предпочтителен (лучше всего поддерживается, включая Safari/iOS).  
Дополнительно можно (или вместо) положить:

- `public/sfx/correct.ogg`
- `public/sfx/wrong.ogg`

После этого звуки будут проигрываться:

- во всех **drill**-тренажёрах (mental math / уравнения / таблица умножения),
- во всех **столбиковых** тренажёрах.

Если звуки не играют в браузере с первого раза — это нормально: браузер может блокировать аудио до первого пользовательского действия.
