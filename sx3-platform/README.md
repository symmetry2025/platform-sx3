## sx3-platform

Новая чистая платформа **SX3** (рядом с SX2), реализуется по этапам из `../docs/SX3_MASTER_PLAN.md`.

### Быстрый старт

- `./sx3 --help`

Если нужно:

- `./scripts/setup-local-node.sh`
- `source ./.tools/env.sh`
- `corepack enable && corepack prepare pnpm@latest --activate`
- `pnpm install`
