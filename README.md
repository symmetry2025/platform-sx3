## Symmetry Workspace (root)

Цель: единый корень для работы в Cursor, где рядом лежат:
- **`sx2-platform/`**: ссылка на текущую рабочую платформу (без изменений).
- **`sx3-platform/`**: новая чистая платформа SX3 (разрабатываем параллельно).
- **`projects/`**: будущие проекты (будем переносить постепенно).
- **`legacy-worktrees/`**: ссылка на текущие worktrees (`account-wt`, `game-wt`, `platform-wt`, ...).

### Как пользоваться в Cursor

- Открывайте workspace как папку `Symmetry Workspace/` (корень этого репозитория).
- Либо откройте файл `SYMMETRY_WORKSPACE.code-workspace`.
